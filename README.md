# AI-Cockpit: The Blueprint Life Operating System

[日本語](#japanese-version)

**AI-Cockpit** is a unified workspace designed to manage all domains of life—business, operations, learning, and brand—as a single, cohesive "Life Operating System."

## 📂 Project Navigation

The repository is organized into **Domains**:

- **[🖥️ Cockpit Core](file:///Users/ryosaigo/AI-Cockpit/domains/cockpit-core)**: The Master Dashboard. Start here.
- **[💼 Consulting](file:///Users/ryosaigo/AI-Cockpit/domains/consulting)**: Business strategy & ventures.
- **[⚙️ Operations](file:///Users/ryosaigo/AI-Cockpit/domains/operations)**: Admin, finance, life management.
- **[🎓 Learning](file:///Users/ryosaigo/AI-Cockpit/domains/learning)**: Studies & skill acquisition.
- **[🎨 Portfolio](file:///Users/ryosaigo/AI-Cockpit/domains/portfolio)**: Personal brand & public identity.

See [docs/STRUCTURE.md](file:///Users/ryosaigo/AI-Cockpit/docs/STRUCTURE.md) for detailed architecture.

## 🚀 Getting Started

To launch the Master Control Dashboard:

```bash
cd domains/cockpit-core
npm install
npm run dev
```

The dashboard will be available at `http://localhost:4321/`.
