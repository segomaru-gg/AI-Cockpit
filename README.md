# AI-Cockpit: The Blueprint Life Operating System

This is a unified workspace designed to manage all domains of life using the **Blueprint Protocol**. It integrates business, operations, personal learning, and brand evolution into a single, cohesive "Life Operating System."

## 🌌 Core Concept: Life OS

The **AI-Cockpit** is more than just a code repository; it is a tactical control center. By standardizing project metadata and task management, it allows for a high-level overview of progress across multiple independent domains.

- **[BluePrint/](file:///Users/ryosaigo/AI-Cockpit/BluePrint)**: The Master Control Dashboard. An Astro-based interface that aggregates tasks and journal entries from all sibling projects.

## 📜 Blueprint Protocol v1.0

Every project in this workspace follows a standardized structure to ensure compatibility with the Master Cockpit:

1. **`blueprint.yaml`**: Metadata (Name, ID, Category, Status).
2. **`tasks/`**: Domain-specific tasks stored as Markdown with technical frontmatter.
3. **`journal/`**: Immutable logs of decisions and milestones.
4. **`.cursorrules`**: Global agent instructions for protocol enforcement.

## 🚀 Getting Started

To launch the Master Control Dashboard:

```bash
cd BluePrint
npm install
npm run dev
```

The dashboard will be available at `http://localhost:4322/`.
