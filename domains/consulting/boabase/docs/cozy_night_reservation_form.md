# CozyNight - Sangria edition 予約フォーム構成案

## 2. 設問項目（必須/任意）

1. **代表者様のお名前（フルネーム）** 【必須】
   - 形式：記述式
   - 説明：当日エントランスにて受付のお名前を確認させていただきます。

2. **ご予約人数** 【必須】
   - 形式：プルダウンまたはラジオボタン
   - 選択肢：1名 / 2名 / 3名 / 4名 / 5名以上（詳細は備考欄へ）

3. **ご来店予定時間（目安で構いません）** 【必須】
   - 形式：プルダウン
   - 選択肢：18:00〜 / 19:00〜 / 20:00〜 / 21:00〜 / 22:00〜
   - 説明：混雑緩和の参考にさせていただきます。途中入場・退場は自由です。

4. **何を見てこのイベントを知りましたか？（来店動機）** 【必須】
   - 形式：チェックボックス または 複数選択
   - 選択肢：Instagramの投稿 / スタッフのストーリー / 友人・知人の紹介 / 過去のイベントに参加した / その他
   - 説明：今後のイベント運営の参考にお聞かせください。

5. **ご連絡先（Instagramアカウント名 または メールアドレス）** 【任意】
   - 形式：記述式
   - 説明：万が一、イベント内容の変更や緊急のご連絡がある場合のみ使用します。（例：@boabase_rokko）

5. **確認事項（年齢・撮影等に関する同意）** 【必須】
   - 形式：チェックボックス
   - 選択肢：
     - [ ] 参加者全員が20歳以上であり、当日身分証を提示することに同意します。
     - [ ] 当日写真や動画の撮影を行います。SNS等で画像が使用される場合があることに同意します。

6. **ご質問やメッセージがあればどうぞ！** 【任意】
   - 形式：段落（長文回答）

## 4. 運用・管理ルール (Sego/Caesar共有事項)
- **通知設定**: フォームに回答があった際に、BoaBASEのLINEグループまたは指定のメールアドレスに通知が飛ぶように設定する。
- **予約リストの連動**: Googleスプレッドシートに回答を自動出力し、当日のエントランス（受付）スタッフがスマホやiPadで閲覧・チェックマークを付けられるようにする。
- **SNS告知時の必須確認**: プロフィールリンクのURLが確実に更新されているか（リンクツリー等の場合は、古いリンクや他フォームになっていないか）をテストする。
