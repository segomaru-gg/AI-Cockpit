**背景**:
外出先からプロジェクトの状況を確認したいが、現状のPC向けレイアウトではスマホで閲覧困難。
「いつでもどこでもコックピット」を実現するための第一歩として、モバイル表示を最適化する。

**要件**:
- スマホ（<768px）でアクセスした際にレイアウトが崩れないこと
- Activity Grid が画面外にはみ出さず、スクロール等で閲覧可能であること
- プロジェクトカードが全幅（1カラム）で表示されること
- タッチ操作で情報が見切れないこと

**完了条件**:
- [x] `ActivityGrid.astro` に横スクロール機能実装
- [x] `index.astro` にモバイル用メディアクエリ追加（縦積みレイアウト）

- [x] 実機相当の解像度での表示確認
