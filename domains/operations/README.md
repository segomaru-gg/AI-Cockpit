# BoaBASE プロジェクト

> **「CHILL & Community」の創造**

BoaBASEは、シーシャを通じたアート、音楽、空間の融合を目指すプロジェクトです。
現在は「現場運営（店長フェーズ）」から「文化的経営（監督フェーズ）」への重要な移行期にあります。

## プロジェクトの変遷

### フェーズ1：店長業務（〜2026.03）
現場の「店長」として、六甲店・元町店の運営を統括し、文化の基礎を築くフェーズ。
- **重点項目:** 現場オペレーション、コミュニティ形成、文化の言語化・インフラ整備。

### フェーズ2：文化経営監督（2026.04〜）
外部パートナー（Aeternum）として、ブランドの思想と構造を統括・支援するフェーズ。
- **重点項目:** ガバナンス、ブランド翻訳（メディア発信）、戦略的キュレーション。
